# AppDemo

## Descripció

Projecte web desenvolupat amb HTML, CSS i JavaScript.

## Funcionalitats

* Disseny responsive
* Bones pràctiques d'accessibilitat
* Validació de formularis
* Mostrar i ocultar informació amb JavaScript
* Esdeveniments d'usuari

## Tecnologies utilitzades

* HTML5
* CSS3
* JavaScript

## Instruccions d'ús

1. Descarregar els fitxers del projecte.
2. Obrir `index.html` amb un navegador web.
3. Provar les funcionalitats interactives.
